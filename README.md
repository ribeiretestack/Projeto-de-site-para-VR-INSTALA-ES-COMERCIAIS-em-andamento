# 🌐 Projeto de Site: VR Instalações Comerciais (Em andamento)

Este é um projeto focado em criar uma presença digital robusta para a **VR Instalações Comerciais**. O site está sendo desenvolvido com foco em performance e uma experiência de usuário (UI/UX) intuitiva para facilitar o contato de clientes e a visualização de serviços.

## 🛠 Tecnologias e Ferramentas
- **Design & Prototipagem:** Figma (UI/UX)
- **Desenvolvimento:** HTML5, CSS3 e JavaScript
- **Integração:** Consumo de APIs para dinamismo

## 🚀 Status do Projeto
O projeto está atualmente **em desenvolvimento**. Estou focando no refinamento das animações de interface e na integração final com serviços de contato.
